# Naham Admin App

لوحة إدارة Naham — بنفس ثيم وتصميم تطبيق TC (العميل والطباخ).

## المتطلبات

- Flutter SDK >=3.0.0
- نفس مشروع Firebase المستخدم في TC (انسخ `firebase_options.dart` من `../tc/lib/` أو شغّل `flutterfire configure` داخل مجلد admin)

## التشغيل

```bash
cd admin
flutter pub get
# انسخ assets/images/logo.png من مشروع tc إن وُجد
flutter run
```

## الثيم والألوان

يُستخدم نفس الثيم من TC دون إضافات:

- **Primary:** #9B7EC8  
- **Secondary:** #7B5EA7  
- **Card:** #E8E4F0  
- **Bottom nav:** #C4B0E8  
- **Background:** #F5F0FF  

الخط: Inter (Google Fonts). نصف قطر الزوايا والظلال كما في `AppDesignSystem`.

## الشاشات

1. **تسجيل الدخول** — بريد وكلمة مرور (Firebase Auth)
2. **لوحة التحكم** — إحصائيات وإجراءات سريعة
3. **إدارة المستخدمين** — تبويبات: الطباخون / العملاء
4. **إدارة الطلبات** — جدول + بحث
5. **إدارة النقد** — موافقة مدفوعات
6. **التفتيش الصحي** — بدء مكالمة فيديو عشوائية (Agora)
7. **التحليلات** — رسوم بيانية (fl_chart)
8. **الدردشة والدعم** — قائمة محادثات + غرفة دردشة
9. **الموافقات المعلقة** — الموافقة/الرفض للطباخين (Firestore)
10. **الإشعارات** — قائمة إشعارات
11. **مكالمة الفيديو** — واجهة Agora (ربط القناة لاحقاً)

## RTL

التطبيق يعمل من اليمين لليسار (العربية) عبر `Directionality(textDirection: TextDirection.rtl)`.

## Firestore

نفس المسارات المستخدمة في TC:

- `users` — المستخدمون (دور، حالة موافقة الطباخ، إلخ)
- `orders` — الطلبات

## Agora

شاشة التفتيش الصحي تفتح شاشة فيديو؛ التكامل الكامل مع Agora (join/leave channel، token) يُضاف عند جاهزية الباكند.
