# هوية مطابخ الديمو (سعودي — نهام)

مرجع لأسماء واقعية للمشروع والتخرج: **بيت + ثقة + تنوع** (بدون شعور placeholder).

---

## توزيع مقترح (لا تُكثري نمط «مطبخ أم …» وحده)

| النوع | العدد التقريبي | أمثلة |
|--------|----------------|--------|
| **مطبخ أم + اسم** | 5 | مطبخ أم فيصل، مطبخ أم ناصر، مطبخ أم خالد … |
| **أسماء لطيفة / سفرة** | 4 | وردة الوردات، سفرة الهنا، بيت المذاق، حامض حلو |
| **تقليدي / نجد** | 3 | سفرة نجد، طعم الأولين، نفس الوالدة |
| **حديث (startup)** | 1–2 فقط | Dar Kitchen، TasteHub — للتنوع |

## لمسة واجهة (عرض البطاقة)

بدل سطر واحد إنجليزي جاف:

```
مطبخ أم فيصل
📍 النرجس، الرياض
🟢 متاح الآن
```

(الحالة من `is_online` + المدينة/الحي من `kitchen_city` أو العنوان.)

## ربط الاسم بالأكل (للمنطق في الديمو)

| المطبخ | اتجاه الأطباق في القائمة |
|--------|---------------------------|
| سفرة نجد | كبسة، جريش، حنيني، قرسان |
| حامض حلو | لقيمات، كنافة، بسبوسة |
| ذوق ومذاق | تشكيلة مختلطة واضحة |
| طعم الأولين | مشاوي، قدرة، رز بسمتي |
| نفس الوالدة | حريص، مرقوق، أطباق بيت |

## توسعة لاحقة (اختياري)

- أوصاف عربية قصيرة لكل طبق + صورة غلاف لكل مطبخ (App Store style).
- ملف seed SQL كامل لـ 12 مطبخ: يُبنى بعد تثبيت الإيميلات في `auth.users`.

---

## ملف الـ seed الحالي

`supabase_graduation_demo_seed.sql` يحدّث أسماء المطابخ المعتمدة (مطبخ أم فيصل، وردة الوردات، مطبخ أم خالد) في `chef_profiles` ونصوص الطلبات/الإشعارات المرتبطة بالديمو.
